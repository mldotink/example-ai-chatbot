# AI Chatbot

A simple AI chatbot powered by OpenAI GPT-4o-mini, ready to deploy.

## Setup

```bash
npm install

# Set your OpenAI API key
cp .env.example .env
# Edit .env with your actual API key

node server.js
```

Open [http://localhost:3000](http://localhost:3000) to start chatting.

## Deploy with Ink

Connect the [Ink MCP server](https://docs.deployink.com/quick-start) to your AI agent and prompt:

> Deploy this app with Ink. Set the OPENAI_API_KEY environment variable.
